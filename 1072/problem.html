Hogwarts Math

George Weasley is terrified of his math test because of one problem in particular. Fortunately, laptops are allowed in Hogwarts, and he needs your help to write a program to solve his problem.

George is given a very large number. He is tasked with continuously dividing this number by one of the prime numbers: 2, 3, 5, or 7, until the quotient becomes 0. He must record the number of times the division was made. Your task is to write a program to help him with this.

  • Your program should prompt the user to enter a large number $n$.
  • It should then prompt the user to enter one of the prime numbers $p$: 2, 3, 5, or 7.
  • The program should divide the large number by the chosen prime number continuously until the quotient is 0 and keep track of the number of divisions made.
  • Finally, the program should output the number of times the division was made.

Constraints

The large number $n$ is a positive integer where $1 \leq n \leq 10000$
The prime number $p$ will be either 2, 3, 5, or 7.

Sample Run 1

Here is a sample run of your program. Please note that the bold text represents user inputs. Please note that the prompts must match the following exactly to get maximum points. 

Enter a large number: 100
Enter a prime number (2, 3, 5, or 7): 2
Number of divisions made:  7
Explanation:
100 is divided by 2 to get 50 (1 division)
50 is divided by 2 to get 25 (1 division)
25 is divided by 2 to get 12 (1 division, remainder is discarded)
12 is divided by 2 to get 6 (1 division)
6 is divided by 2 to get 3 (1 division)
3 is divided by 2 to get 1 (1 division, remainder is discarded)
1 is divided by 2 to get 0 (1 division, remainder is discarded).
In total, there are 7 divisions.
