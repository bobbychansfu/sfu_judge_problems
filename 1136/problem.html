Counting

In combinatorial mathematics, counting problems can often be categorized based on whether they involve arrangements (order matters) or selections (order does not matter), and whether repetitions are allowed. Given two integers $n$ and $r$, you are required to write a function to calculate the number of ways to select $r$ items from $n$ items under different conditions. The function should take four parameters:

  1. order: A boolean indicating whether the order of selection matters.
  2. repeat: A boolean indicating whether repetitions are allowed.
  3. n: The total number of items.
  4. r: The number of items to select.

Write a function called counting_formula(order, repeat, n, r)
Parameters:

  • order (boolean): True if the order matters, False otherwise.
  • repeat (boolean): True if repetition is allowed, False otherwise.
  • n (integer): The total number of items.
  • r (integer): The number of items to select.
Returns: The number of ways to select $r$ items from $n$ items based on the specified conditions.

Calculation Rules

  • Order matters, repetition allowed: $n^r$
  • Order matters, no repetition: $\frac{n!}{(n-r)!}$
  • Order does not matter, repetition allowed: $\frac{(n+r-1)!}{r!(n-1)!}$
  • Order does not matter, no repetition: $\binom{n}{r}$ or $\frac{n!}{r!(n-r)!}$

Input Format

The function will be called with four arguments: order, repeat, n, and r.

Output Format

The function should return an integer representing the number of ways to select $r$ items from $n$ items based on the specified conditions.
