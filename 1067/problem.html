What BMI am I?

Write a program to calculate the body mass index (BMI) for adults aged 18-65. The formula is as follows:
$BMI = w / h^2$

The BMI categories are as follows:

The program should take as input:

  • Weight $w$ in kg
  • Height $h$ in m
And output the BMI category.

Category BMI Value
Severe Thinness < 16
Moderate Thinness [16, 17)
Mild Thinness [17,18)
Normal [18,25)
Overweight [25,30)
Case I Obesity [30,35)
Case II Obesity [35,40)
Case III Obesity ≥ 40
