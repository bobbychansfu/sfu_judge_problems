File Fun

This is to teach problem-writers and makers how to use files!

Input

Your program should read a list of english five-letter words from words.txt (which will stay consistent over all test cases), a data word from data.txt, and an input word from standard input (stdin).

All words will consist of five lower-case characters. Note that there may be whitespace before or after any of the words.

Output

Output three lines to stdout, indicating in order with true or false whether:

  • The input word is in the word list
  • The data word is in the word list
  • The input word and data word are the same

Sample 1

stdin

water

data.txt

abcde

stdout

true false false
