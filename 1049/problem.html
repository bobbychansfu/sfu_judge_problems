John is building a swimming pool. Before he starts, he wants to imagine what it would look like. He would like you to help him write a program to draw his swimming pool. Given a width $w$ and a length $l$, draw the resulting pool.

Input

Integers $w$ and $l$ ($1 \leq w,l \leq 27$), where $w$ is the width and $l$ is the length of the pool.

Output

Draw the pool.
