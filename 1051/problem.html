Translating Romans

Bob is trying to teach his son Jacob to read Roman Numerals, but he's not very motivated. So, he's asked you to help by writing a program to convert Roman Numerals to integers.

Roman numerals are a way of writing numbers using seven symbols: I, V, X, L, C, D, and M. Each symbol has a specific value, as shown below:

SymbolValue
I1
V5
X10
L50
C100
D500
M1000

You use combinations of these symbols to write a number in Roman numerals. For example, the number 2 is written as II in Roman numerals, which is simply two ones added together. The number 12 is written as XII, which is just 10 plus 2. And the number 27 is written as XXVII, which is 20 plus 5 plus 2.

When writing Roman numerals, you usually write the largest symbols first and the smallest symbols last, from left to right. However, there are some exceptions. For instance, instead of writing IIII for the number 4, you write IV, which means 5 minus 1. Similarly, instead of writing VIIII for the number 9, you write IX, which means 10 minus 1. There are only six instances where subtraction is used like this:

  • For 4, we put I (1) before V (5)
  • For 9, we put I (1) before X (10)
  • For 4, we put X (10) before L (50)
  • For 9, we put X (10) before C (100)
  • For 400, we put C (100) before D (500)
  • For 900, we put C (100) before M (1000)
