John's Bank

John’s National Bank has a straightforward interest calculation algorithm. The accrued total amount (total amount of money you have) is calculated using the given principal amount and the time period in years using the following equation
$total = principle * (1.0 + rateOfInterest * years) $

The rateOfInterest is given by the following table:

Principle Amount Rate of Interest
0.00 ≤ p < 1000.00 0.025
1000.00 ≤ p < 10,000.00 0.02
10,000.00 ≤ p < 100,000.00 0.015
p ≥ 100,000.00 0.01

Help John to write a program that asks the user for the principal amount $p$ and the number of years $y$, then prints the total $t$ after $y$. If the principle or the number of years is less than 0, print “not possible.”

Input

First input is a floating-point number $p$, the principal. The second input is a floating-point number $y$, the total number of years.

Output

The number $t$ rounded to two decimal places, the total amount after $y$ years.

Sample Run 1

Here is a sample run of your program. Please note that the bold text represents user inputs. Please note that the prompts must match the following exactly to get maximum points. 

    Please enter the principal: 23000.99
    Please enter the years: 2.3
    3794.52
