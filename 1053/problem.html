Data Domains

A data domain is a set of values that can be assigned to a particular data element or attribute, which helps ensure data integrity and consistency in a database.

A specialization of the data domains problem is when we need to determine whether a list of numbers falls within a specified range of integers. Write a program that reads a list of integers separated by a single space in the first line. In the next two lines, the program should read two integers $i$ and $j$, where $i$ < $j$. The program should then determine whether all the integers in the list fall within the range of $i$ and $j$ (inclusive) and print 'yes' or 'no' accordingly.

Input

The first line will contain $n$ integers separated by whitespaces. The second contains the variables $i$ and $j$ separated by a whitespace.

Output

the string 'yes' or 'no'

Sample Run 1

Here is a sample run of your program. Please note that the bold text represents user inputs. Please note that there are no prompts in this question. 

4 5 6 7 3 9
3 10
yes

Sample Run 2

4 1 6 7 2 9 5
3 10
no
