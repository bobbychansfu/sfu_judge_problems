Repetitions

In the microscopic metropolis of Geneopolis, where the DNA strands twist and twirl like city streets, the annual "Festival of the Spirals" was the day when the longest single-character DNA sequence was celebrated. Ada, a young scientist, has asked you to create a program to output the number of the longest sequence of characters in a DNA strand.

Given a DNA sequence: a string consisting of characters A, C, G, and T. Your task is to find the longest repetition in the sequence. This is a maximum-length substring containing only one type of character.

Input

The only input line contains a string of $n$ characters.

Output

Print one integer: the length of the longest repetition.

Constraints

$1 \le n \le 10^6$

Sample Input

ATTCGGGA

Sample Output

3
