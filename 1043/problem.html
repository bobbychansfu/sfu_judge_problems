Fencing Madness

Bob is fencing his home. He knows he must put a fence post up every 4 meters starting from 1 (i.e. 1, 5, 9, 13, etc.). Given a starting point and an ending point, Bob would like to know where to put the fence posts. Write a program asking the user for a range [x,y] and prints out all of the locations where Bob must put a fence post.

Input

Prompt the user to give the $x$ and $y$, values

Output

All the locations between $x$ and $y$ where Bob must put a fence post.
