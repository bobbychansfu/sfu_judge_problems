Marking Attendance

A kindergarten teacher needs to mark attendance but faces a challenge: the children are active and do not stay seated during roll call. To manage this, the teacher decides to write a Python program to record each student's name without repetition, as the attendance sheet has limited space.

Your task is to write a Python program that allows the teacher to input student names one at a time. The teacher will enter 'n' to finish the input process. The program should then display all unique student names in alphabetical order, separated by commas.
