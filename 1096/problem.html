Actual Addition

Bob and Alice are testing an online Judge to be used at SFU for various things! They really need your help with testing and other things, though!

Please help them by solving this simple problem, used for testing. You'll get rewarded handsomely!

Input

The first line of input will contain a number N ($2 \leq N \leq 100$), the number of numbers to be summed.

The next line will contain $N$ numbers $a_1, a_2, \dots, a_N$ ($1 \leq a_i \leq 10^5$).

This input will be read through a text file: input.txt. Please refer to the starter code.

Output

Output the sum of the $N$ numbers: $\displaystyle \sum_{i=1}^N a_i$.

Sample Input

5
1 2 3 4 5

Sample Output

15
