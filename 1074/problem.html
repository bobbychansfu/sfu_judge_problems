Grace Exams

Grace is a student preparing for her upcoming exams and would like a program to determine if she has prepared well enough. Write a program to help her with the following requirements:

The program should ask the user to enter the number of days she has left to study before the exam (referred to as D-Day).
It should then ask for the total number of mock exams she needs to pass before D-Day.
For each day leading up to D-Day, the program should ask the user how many mock exams she completed that day.

If she completes at least the required number of mock exams by D-Day, the program should output: "D-Day! Your exam will go as planned".
If she completes fewer mock exams than required by D-Day, the program should output: "D-Day! Still, hope for the best."

Sample Run 1

Here is a sample run of your program. Please note that the bold text represents user inputs. Please note that the prompts must match the following exactly to get maximum points. 

Enter the number of days left until D-Day: 4
Enter the number of mock exams needed to pass: 5
How many mock exams did you complete on day 1? 1
How many mock exams did you complete on day 2? 2
How many mock exams did you complete on day 3? 1
How many mock exams did you complete on day 4? 1
D-Day! Your exam will go as planned.
