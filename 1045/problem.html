Sorry Folly

Bob trying to teach his son Jacob to say sorry. But Jacob is so stubborn that he often takes many iterations to give in. Help Bob write a "sorryBot" that stops asking only when an apology is detected.

Input

Prompt the user to give an apology.

Output

Output "Good boy." when the user enters the string "sorry"

Sample Run 1

Here is a sample run of your program. Please note that the bold text represents user inputs. Please note that the prompts must match exactly to receive a maximum score. 

What do you say? no
What do you say? arrgh
What do you say? blah blah
What do you say? NO!!!!
What do you say? sorry
Good boy.
