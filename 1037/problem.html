Country Capital

Alice wants to create some practice materials for her geography class. She will enter the countries and their capital separated by a space in the console. Help her write a program to print the corresponding information into a well-formatted sentence.

Input

One line with the country name followed by the capital. You may assume the capital city names may contain spaces but country names may not.

Output

A single line that prints out the well-formatted sentence with the input information.

Sample Run 1

Here is a sample run of your program. Please note that the bold text represents user inputs. Please note there are no prompts for user input. 

Canada Ottawa 
Ottawa is Canada's capital city

Sample Run 2

U.S.A. Washington D.C. 
Washington D.C. is U.S.A.'s capital city
