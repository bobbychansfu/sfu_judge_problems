Age Analytics

Kelly, a curious and enthusiastic computer science student, is fascinated by data analytics. She is particularly interested in understanding the age dynamics of her class. To delve into this, she decides to write a program that not only finds the average age of her classmates but also identifies the oldest student(s).

Your task is to help Kelly by writing this program. The program should:

  • Take as input the number of students in the class.
  • Receive each student's name followed by their age (one student per line).
  • Calculate and print the average age of the students, rounded to two decimal places.
  • Identify the oldest student(s) and print their names in lexicographical order.

Input Format

The first line contains an integer, $n$, the number of students in the class.
The next $n$ lines each contain a string and an integer, representing the name of the student and their age, respectively.

Output

The first line should display the average age of the class, rounded to two decimal places.
The following line(s) should list the name(s) of the oldest student(s), each name on a new line, in lexicographical order.

Constraints

  • $1 \leq n \leq 100$
  • $1 \leq Age \leq 100$
  • Names consist only of alphabetic characters and are not longer than 20 characters.

Sample Input 1

5
Alice 23
Bob 25
Charlie 25
Diana 22
Eve 24

Sample Output 1

23.80
Bob
Charlie

Sample Input 2

4
Liam 18
Noah 20
Olivia 18
Emma 20

Sample Output 2

19.00
Emma
Noah
