Floating Jacob

Little Jacob is learning floating point numbers. His dad Bob, being lazier than most, would like to teach his son by writing a program to report the relativity of two floating-point numbers entered from the console. Can you help Bob write this program so he can return to watching Squid Games?

Input

Prompt the user by asking for numbers x and y.

Output

A single line that prints out the relativity of these numbers. Print

  • "x and y are the same" if x and y are equal. Replace x and y in the string with their values. Otherwise,
  • "a is less than b", where is a is the min of x and y, and b is the max of x and y
