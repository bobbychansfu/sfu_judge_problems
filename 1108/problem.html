Unlock the Tower

In a distant digital kingdom, there is a tower with a magical number lock programmed with a recursive sequence. The lock displays a number and asks for the next number in a special sequence. This sequence follows a rule: each number is the sum of the previous three numbers, starting with 1, 1, and 2.

As the kingdom's most skilled programmer, your task is to unlock this tower by writing a recursive function in Python to find the $n^(th)$ number in this sequence, where $n$ is the position displayed by the lock.

Input Format

A single integer $n$ representing the position in the sequence.

Output

The $n^(th)$ number in the sequence.

Constraints

  • $1 \leq n \leq 50$

Sample Input 1

7

Sample Output 1

24

Explanation: The first seven numbers in the sequence are 1, 1, 2, 4, 7, 13, 24. The seventh number is 24.
