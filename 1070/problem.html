Fire Escape

You are trapped in a building on fire and must make the correct choices to escape. You will be prompted to answer the following questions:

  1. (1) Take many things or (2) Take nothing?
  2. (1) Stay in the room or (2) Leave the room?
  3. (1) Walk or (2) Crawl?
  4. (1) Take the elevator or (2) Take the stairs?

For each question, if the user chooses option (1), inform the user that they have not survived and restart the questionnaire from the first question. If the user chooses option (2), commend them and proceed to the next question.

If the user correctly chooses option (2) for all four questions, notify them that they have successfully escaped the fire and terminate the program.

The user will have a maximum of four attempts to escape; after which, the program will end.

Write a program that continues to prompt the user for moves until they either win or lose the game. The user should enter one of the three possible moves ('step', 'dig', 'rest') in each turn. The moves are NOT case sensitive, 'DIG' equals 'Dig' equals 'diG'.

Requirements

  • Prompt the user to answer each question in sequence.
  • Validate the user's input and handle any invalid responses gracefully, re-prompting the user as necessary.
  • If the user chooses option (1) for any question, restart the questionnaire.
  • If the user chooses option (2) for all questions, congratulate them and end the program.
  • Limit the user to four attempts to escape. If the user fails to escape after four attempts, terminate the program with an appropriate message.
