Ghost Month

In Malaysia, during the ghost month, there are several taboos that elders often remind children about. Listed below are a few of these taboos:

  1. Going out during midnight.
  2. Taking photos at night as one pleases.
  3. Drying clothes outside at night.
  4. Going near water.
  5. Patting people on the shoulder.

Write a program that asks the user whether they adhere to these taboos.

  • For each taboo, the program should prompt the user with a yes/no question.
  • If the user responds with anything other than 'yes' or 'no', the program should re-prompt the user for that specific taboo.
  • If the user responds 'yes' to any of the taboos, the program should terminate immediately and output: 'Uh oh! Ghosts are coming for you!'
  • If the user responds 'no' to all the taboos, the program should output: 'You are the taboo master!'

Sample Run 1

Here is a sample run of your program. Please note that the bold text represents user inputs. Please note that the prompts must match the following exactly to get maximum points. 

Do you usually go out during midnight (yes/no): no
Do you take photos as you like at night (yes/no): No
Do you usually whistle at night (yes/no): Yes
Uh oh! Ghosts are coming for you!
