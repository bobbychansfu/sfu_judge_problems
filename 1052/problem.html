String Shuffler

Joe has an interesting job as a "string shuffler." As a string shuffler, he takes two strings and makes a new list of all the possible combinations of their characters while keeping their original order. For example, if Bob is given the strings "ABC" and "ACB," the list of shuffled strings are:
ACBABC, AABCBC, ACABCB, ABCACB, AACBBC, ABACCB, ACABBC, ABACBC, AACBCB, AABCCB

Another example, if Joe is given the strings "ABC" and "DE," the shuffled strings would be
ABCDE, ABDCE, ADBCE, DABCE, DABEC, DAEBC, DEABC

Joe needs some help writing a function called "shuffleAll(X,Y)" that takes two input strings X and Y and returns a shuffled list of strings from X and Y. To do this, he might need to write some other functions to help him, which could be recursive.

To complete this task, you can download the provided sample code and write the shuffleAll(X,Y) function. Do not change anything else in the code.
shuffleAll("ABC","ACB") returns [ 'ACBABC', 'AABCBC', 'ACABCB', 'ABCACB', 'AACBBC', 'ABACCB', 'ACABBC', 'ABACBC', 'AACBCB', 'AABCCB' ] in any order and without duplicates.
shuffleAll("ABC","DE") returns [ 'ABCDE', 'ABDCE', 'ABDEC', 'ADBCE', 'ADBEC', 'ADEBC', 'DABCE', 'DABEC', 'DAEBC', 'DEABC' ] in any order and without duplicates.
