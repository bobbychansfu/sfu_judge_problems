ParentBot

It is a known fact that aliens periodically abduct people to do small experiments on them before returning them home a few days later. During this time, they will replace the real human with a life-like robot that resembles their humans. Zergiug has been tasked with writing the logic for a ParentBot, who will ask the kids four questions. Every time answer is ‘yes’, the points increase. The questions are:

  • Did you eat?
  • Did you study?
  • Did you do your laundry?
  • Did you call grandma?
At the end, ParentBot will reply as follows.
  • 0 points: We need to talk.
  • 1-2 points: Ok.
  • 3-4 points: Good!
Please help Zerguig write this program.

Input

On four lines, the inputs will be either 'yes' or some other string.

Output

Print ParentBots answer depending on how many times 'yes' was answered

Sample Run 1

Here is a sample run of your program. Please note that the bold text represents user inputs. Please note that the prompts must match the following exactly to get maximum points. 

    Did you eat? yes
    Did you study? blah
    Did you do your laundry? yes
    Did you call grandma? nah
    Ok.

Sample Run 2

    Did you eat? no
    Did you study? blah
    Did you do your laundry? nope
    Did you call grandma? oops
    We need to talk.
