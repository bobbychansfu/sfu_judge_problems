Merlin and the Enchanted Stones

Long ago, a talented young wizard named Merlin was known for his ability to solve complex puzzles using magical Python spells. The King has ordered him to solve a critical puzzle: In the heart of the kingdom lies a magical forest filled with enchanted stones vital for the safety of the realm. This forest is structured as a grid, with each cell possibly containing an enchanted stone.

Your task is to help Merlin identify the coordinates of all the enchanted stones in the grid.

Input

The first line contains two integers $r$ and $c$ ($1 \leq r, c \leq 1000$), separated by a space, indicating the number of rows and columns in the grid, respectively. The next $r$ lines describe the grid. Each of these lines contains $c$ characters separated by a space, representing a row of the grid. Each character is either an 'X' (representing an enchanted stone) or an 'O' (representing an empty space). Note: there are no user prompts for this question.

Output

Output all the coordinates of the 'X's in the grid, representing the enchanted stones. Each coordinate should be represented by a pair (row, column), and each pair should be on a new line. The rows and columns are 1-indexed. The coordinates should be listed in row-major order (left to right, top to bottom). If there are no stone, output "None".

Sample Input 1

3 3
O X O
X O O
O O X

Sample Output 1

1 2
2 1
3 3

Sample Input 2

3 3
O O O
O O O
O O O

Sample Output 2

None

Explanation: In this example, there are three enchanted stones located at positions (1, 2), (2, 1), and (3, 3) on the grid.
